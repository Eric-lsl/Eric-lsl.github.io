if-else语句

if-else语句也是C++中的判断语句，如果S为真（非零），则执行语句1，否则执行语句2。
也就是说，程序运行到if-else语句，必然要执行其中一个语句，且只执行其中一个语句。

语句1和语句2可以是单个语句，也可以是复合语句。

if-else-if语句

如果S1为真（非零），则执行语句1；否则判断S2，如果S2为真，则执行语句2，否则执行语句3。
也就是说，程序运行到if-else-if语句，必然要执行其中一个语句，且只执行其中一个语句。

语句1、语句2、语句3可以是单个语句，也可以是复合语句。
